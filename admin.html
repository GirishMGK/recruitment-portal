Firebase Setup Not Connected
Complete this once to enable cross-device data sync for candidates and admin
1
Go to console.firebase.google.com and sign in with your Google account
2
Click "Create a project" → Enter project name (e.g., gk-recruitment) → Continue → Disable Analytics → Create
3
In the left menu, click "Build → Realtime Database" → Click "Create Database" → Choose Asia-Pacific (Singapore) → Select "Start in test mode" → Enable
4
Click the ⚙️ gear icon next to "Project Overview" → Project Settings → Scroll to "Your apps" → Click the </> (Web) icon → Register app (any name) → Copy the firebaseConfig object
5
Open both index.html and admin.html in a text editor → Find FIREBASE_CONFIG at the top → Replace the placeholder values with your copied config → Save both files → Re-upload to GitHub Pages
Firebase config is safe to be public — it's designed this way. Security is managed by Firebase Database Rules. The test mode rules expire after 30 days; contact us to set up proper rules before then.
